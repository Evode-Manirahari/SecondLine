🧠 Persistent memory

Recognizes returning callers by number, recalls past orders and constraints. "Same as last time, but no lilies — my mom's allergic" just works.

🛡️ Safety-aware

Allergens on file are enforced at the tool layer — the agent physically cannot add a bouquet a caller is allergic to.

🧾 Owner workflow

Every order becomes a structured task — customer, request, confidence, due time, next action — with an SMS summary.

🙋 Knows its limits

Refunds, complaints, impossible asks → escalates to a human instead of guessing. Never invents prices or policy.

The part that wins

It finds and fixes its own mistakes.

Every call is logged, tested, scored, and used to improve the next version — with real automated test calls (Cekura) and an evaluation harness, not vibes.

  • Failures become fixes. Each failed call → a structured bug report → a prompt patch, validation rule, escalation rule, or memory update → re-run.
  • Real automated test calls. Cekura places live voice calls, scores transcripts, and surfaced a production bug our local tests couldn't — which we fixed and re-verified.
